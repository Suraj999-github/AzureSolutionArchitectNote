artificial intelligence workloads

🧠 AI concepts

AI simulates human capabilities: learning, reasoning, perception, natural language. Machine learning is a subset where models learn from data. Deep learning uses neural networks.

  • Training vs inference
  • Supervised/unsupervised/reinforcement learning
📊 machine learning (Azure ML)

Azure Machine Learning service: manage models, automated ML (AutoML), designer (drag-and-drop). ML pipelines, model registry, endpoints.

  • Regression (predict numeric), classification (predict category), clustering (group unlabeled data).
👁️ computer vision

Azure Cognitive Services – Vision: Computer Vision (image analysis, OCR, tags), Custom Vision (image classification/object detection), Face API (detection/recognition).

  • Optical Character Recognition (OCR) extracts text from images.
📝 natural language processing (NLP)

Language services: Text Analytics (sentiment, key phrases, entities), Translator, LUIS (language understanding), Question Answering (QnA Maker).

  • Named Entity Recognition (NER), sentiment analysis, language detection.
💬 conversational AI

Azure Bot Service – build, connect, deploy bots. Integrate with QnA Maker for FAQ-style answers. Use LUIS for natural language understanding.

  • Channels: Web, Teams, Slack, etc.
⚖️ responsible AI principles

Microsoft's six principles: fairness, reliability & safety, privacy & security, inclusiveness, transparency, accountability. Mitigate bias, ensure interpretability.

  • Model interpretability, fairness assessment tools.
📘 exam focus

· Understand AI workloads and which Azure service to use for each. Know the responsible AI principles and their importance.

📋 key services

  • Azure Machine Learning – ML lifecycle
  • Computer Vision – analyze images
  • Custom Vision – custom classifiers
  • Text Analytics – sentiment, key phrases
  • Translator – language translation
  • LUIS – language understanding
  • QnA Maker – FAQ bots
  • Bot Service – conversational agents

🧠 ML types

  • Regression – predict numeric (price)
  • Classification – predict category (spam/not)
  • Clustering – group unlabeled data
  • Supervised – labeled data
  • Unsupervised – no labels
  • Reinforcement – rewards

⚖️ responsible AI

  • Fairness – avoid bias
  • Reliability & safety – robust
  • Privacy & security – data protection
  • Inclusiveness – diverse users
  • Transparency – explainability
  • Accountability – governance

📝 study tips

  • Match workload to service
  • Know cognitive services categories
  • Understand AutoML
  • Familiar with bot components (QnA, LUIS)
  • Responsible AI principles
